# Retail data cleaning and EDA
## Очистка данных онлайн ритейла и разведывательный анализ (EDA)

Проект направлен на отработку навыков аналитической работы в Python. Так как это первый проект здесь, он посвящен базовой задаче очестке "грязных" данных и, непосредственно, их разведывательному анализу.

Датасет был взят с Kaggle: - https://www.kaggle.com/datasets/nudratabbas/real-world-retail-data-for-data-cleaning-easy
Эти данные были выбраны намеренно, так как позволяют поработать с пропущенными значениями, с некорректными названиями, не одинаковым форматом колонок, с датой, а также с несовместимыми форматами отображения количества (например, "500 g").
